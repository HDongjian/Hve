# Hve

基于Vue.js的移动端UI组件库

## 框架简介

- 编写框架基于vue-cli3.0
- 编写CSS基于CSS预处理器Less
- 支持主题设置，rem布局

## 文档

[Hve：移动端组件库](https://hdongjian.github.io/2019/06/18/hve/)
